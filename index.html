Git: Основи та Використання

Проєкт створив: Король Максим

1. Що таке системи контролю версій?

Системи контролю версій (VCS, англ. Version Control System) — це програмне забезпечення, що допомагає відстежувати зміни у вихідному коді або інших файлах проєкту. Вони дозволяють зберігати історію змін, порівнювати версії файлів, а також забезпечують командну роботу над проектами.

2. Що таке Git?

Git — це розподілена система контролю версій, створена Лінусом Торвальдсом у 2005 році. Вона дозволяє кільком розробникам працювати одночасно над одним проєктом, зберігаючи при цьому історію змін, з можливістю повернення до попередніх версій коду.

3. Основні можливості Git

  • Розподілена структура: кожен користувач має повну копію репозиторію на своєму комп'ютері.
  • Підтримка гілок (branches): дозволяє створювати різні версії проєкту для паралельного розвитку.
  • Система злиття (merge): зручна інтеграція змін з різних гілок.

4. Переваги використання Git

  • Ефективна командна робота: кілька розробників можуть працювати паралельно без конфліктів.
  • Збереження історії змін: кожен коміт документує всі зміни, включно з коментарями.
  • Простота створення резервних копій: кожен клон репозиторію є резервною копією.

5. 10 базових команд роботи в Git

1. git init
Ініціалізує новий репозиторій Git у поточній директорії.
git init
2. git clone
Клонує існуючий репозиторій з віддаленого сервера на локальний комп'ютер.
git clone <url-репозиторію>
3. git status
Виводить інформацію про поточний стан робочої директорії.
git status
4. git add
Додає зміни до індексу.
git add <назва_файлу>
git add . (додає всі зміни)
5. git commit
Фіксує зміни в репозиторії.
git commit -m "Опис змін"
6. git log
Виводить історію комітів.
git log
7. git branch
Створює нову гілку.
git branch <ім'я_гілки>
8. git checkout
Перемикається на іншу гілку або відновлює файли.
git checkout <назва_файлу>
9. git merge
Зливає зміни з іншої гілки.
git merge <ім'я_гілки>
10. git push
Відправляє зміни з локального репозиторію.
git push <ім'я_віддаленого_репозиторію> <ім'я_гілки>
